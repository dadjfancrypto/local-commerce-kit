# Security Policy

Local Commerce Kit is a static front-end template. It does not process payments, authenticate users, or store customer data.

## Reporting Issues

If you find a security issue, please open a GitHub issue with a clear description and reproduction steps. Do not include real secrets, customer data, private keys, or payment details in the report.

## Production Use

Before using this template in production, connect it to a trusted checkout provider, validate orders on a backend, publish a privacy policy, and review all third-party assets.
