# Contributing

Thanks for considering a contribution to Local Commerce Kit.

## Good First Contributions

- Improve README clarity
- Add product catalog examples
- Improve keyboard navigation and focus states
- Add accessibility checks
- Fix responsive layout issues
- Add translations

## Development

This project is intentionally static. Open `index.html` directly or run:

```bash
python -m http.server 8080
```

Then open `http://localhost:8080`.

## Pull Requests

Please keep changes focused and include a short explanation of the user-facing behavior. For UI changes, include before/after screenshots when possible.

Do not add build tooling, payment processing, analytics, or external tracking without opening an issue first.
